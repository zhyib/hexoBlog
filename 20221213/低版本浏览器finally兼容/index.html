低版本浏览器finally兼容

最近生产上遇到了一些问题，报错 .finally() is not a function.

核心问题是浏览器版本过低，Chrome 内核版本 60。查表发现，从 63 才开始支持 finally() 语法。所以需要一点转换组件。

组件

  • 执行 npm install promise.prototype.finally -S 安装，这是官方仓库的为了适配低版本浏览器而开发的组件
